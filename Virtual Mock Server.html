k6 Setup Instructions

To avoid rate limiting issues with external APIs like ReqRes, set up a local mock server:

1. Install JSON Server

npm install -g json-server

2. Create a db.json file

{ "login": { "token": "test123" }, "users": [ { "id": 1, "name": "John Doe", "email": "john@example.com" }, { "id": 2, "name": "Jane Smith", "email": "jane@example.com" } ] }

3. Start the JSON Server

json-server --watch db.json --port 3001

4. Update your k6 script

const API_BASE_URL = 'http://localhost:3001';
